# Machine Learning Projects

This repository contains a collection of **machine learning projects** developed as part of my learning in data science and applied machine learning.  
The projects focus on **end-to-end ML pipelines** including data preprocessing, feature engineering, model training, evaluation, and visualization.

---

##  Tech Stack & Tools
- **Language:** Python  
- **Libraries:** NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn, XGBoost, NLTK  
- **Concepts:** Supervised Learning, NLP, Regression, Classification, Cross-Validation, Hyperparameter Tuning  

---

##  Projects Overview

### 1️. Fake News Detection
**Problem:** Classify news articles as fake or real using Natural Language Processing techniques.

**Approach:**
- Text preprocessing: cleaning, stopword removal (NLTK)
- Stemming using PorterStemmer
- Feature extraction using TF-IDF Vectorization
- Model training using Logistic Regression

**Results:**
- Achieved **98% accuracy** on the test dataset

**Key Concepts:** NLP, TF-IDF, Text Classification, Logistic Regression

---

### 2️. House Price Prediction
**Problem:** Predict house prices based on input features for real estate valuation.

**Approach:**
- Data preprocessing and feature selection
- Model training using XGBoost Regressor
- Performance evaluation using R² score and Mean Absolute Error (MAE)

**Results:**
- **R² Score:** 0.84  
- **MAE:** 0.3  
- Visualized actual vs predicted prices using scatter plots

**Key Concepts:** Regression, Ensemble Learning, Model Evaluation

---

### 3️. Diabetes Prediction
**Problem:** Predict whether a patient is diabetic based on medical parameters.

**Approach:**
- Feature scaling using StandardScaler
- Model training using Support Vector Machine (SVM)
- Model evaluation using accuracy score
- Visualization using a confusion matrix

**Results:**
- Achieved **78% accuracy** on test data

**Key Concepts:** Classification, SVM, Feature Scaling, Confusion Matrix

---

### 4️. Rock vs Mine Detection
**Problem:** Classify sonar signals as either rocks or mines.

**Approach:**
- Implemented multiple ML models:
  - Logistic Regression  
  - Support Vector Machine (SVM)  
  - K-Nearest Neighbors (KNN)
- Compared models using k-fold cross-validation
- Applied GridSearchCV for hyperparameter tuning

**Results:**
- Logistic Regression achieved the highest accuracy after tuning

**Key Concepts:** Model Comparison, Cross-Validation, Hyperparameter Optimization
